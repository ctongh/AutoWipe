# AutoWipe - Privacy Protection Tool
自動清理特定網站的瀏覽紀錄和特定關鍵字的搜尋紀錄。

## 主要功能：
- 清理使用者指定網站的瀏覽記錄
- 清理包含特定關鍵字的瀏覽記錄
- 自動定時清理：實現自動化的歷史記錄清理
- 手動清理按鈕：提供即時清理選項
- 網站/關鍵字管理：讓使用者指定要清理的內容

## 使用方法：
1. 登陸你想刪除的關鍵字，例如：「吉伊」
→ 執行結果：清除所有標題含有「吉伊」的搜尋紀錄

2. 登陸你想刪除的網站連結，例如："https://www.ccu.edu.tw/"
→ 執行結果：清除所有網址含有"https://www.ccu.edu.tw/"的搜尋紀錄
